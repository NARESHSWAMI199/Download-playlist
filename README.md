# Download a full playlist from YouTube at once.

## Modules that are needed for this program
- In python 
```
pip install pytube
```
---------- or -----------
```
pip3 install pytube
```

- For video audio mixer 
  - Mac Os.
  ```
   brew install ffmpeg
  ```
  - Linux 
  ```
  sudo apt install ffmpeg
  ffmpeg -version
  ```
